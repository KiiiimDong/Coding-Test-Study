# [BOJ] 좌표압축


[문제링크](https://www.acmicpc.net/problem/18870)

## 1. 문제 설명
### 1.1 문제요약

Xi의 리스트 내 크기 순서 출력

### 1.2 입출력 예시

- 입력

  ```python
5
2 4 -10 4 -9
  ```

- 출력

  ```python
2 3 0 3 1
  ```


## 2. 문제해결 아이디어
리스트 시간 복잡도 : O(n)
딕셔너리 시간 복잡도 : O(1)

------

2    4 -10 4 -9
2    3   0 3  1

-10 -9   2 4  4
0    1   2 3  3
{-10:0, -9:1, 2:2, 4:3, 4:3}
