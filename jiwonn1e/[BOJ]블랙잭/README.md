# [BOJ] 블랙잭

[문제링크](https://www.acmicpc.net/problem/2798)

## 1. 문제 설명
### 1.1 문제요약

척번째 입력 숫자 : 카드 갯수
두번째 입력 숫자 : 카드의 합
출력 : 두번째 입력 숫자를 넘지않고 가까운 카드의 합

### 1.2 입출력 예시

- 입력

  ```python
5 21
5 6 7 8 9
  ```

- 출력

  ```python
21
  ```


## 2. 문제해결 아이디어

123 124 125
134 135
145

234 235
245
